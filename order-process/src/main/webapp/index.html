Order Processing System

A spring CXF maven spring-boot application based on "Apache CXF Web Service Development" book by packtpub.

  • To run: mvn clean package spring-boot:run
  • WSDL:http://localhost:8080/service/OrderProcess?wsdl
  • Home page for all the services: http://localhost:8080/service/
  • Web Service Client:http://localhost:8080/client
